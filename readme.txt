За основу проекта взят проект "Serving Web Content with Spring MVC"
с сайта https://spring.io/guides/gs/serving-web-content

Проект расширен и к нему был прикручен функционал Spring Securty
Ссылочная структура страниц:
Главная страница
    |_ greeting
    |    |_ greeting2
    |_ greeting2

В качестве страницы с ограниченным доступом из белого списка
выбрана страница greeting2
Доступ к ней разрешён только со страницы greeting, соотвественно
при переходе на неё с главной страницы получаем запрет доступа.
